## Building your own résumé

1. Fork & clone this repository
1. Install dependencies
    ```bash
    $ npm install
    ```
1. Start dev server
    ```bash
    $ npm run dev
    ```
1. Update copy in [`src/data.ts`](src/data.ts)
1. Update icons in [`public/`](public/)
